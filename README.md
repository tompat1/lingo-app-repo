# lingo-app-repo

## Getting started

### 1) Set up the environment

- Install **Git** and verify it is available on your path:
  ```bash
  git --version
  ```
- Install **Node.js (LTS)** if you plan to run or build the app locally:
  ```bash
  node --version
  npm --version
  ```
- Clone the repository and enter the project directory:
  ```bash
  git clone <your-repo-url>
  cd lingo-app-repo
  ```

### 2) Set up Visual Studio Code

- Install **Visual Studio Code** from https://code.visualstudio.com/.
- Open the project:
  ```bash
  code .
  ```
- Recommended extensions (optional, but helpful):
  - ESLint
  - Prettier
  - EditorConfig

### 3) Next steps

Once the environment is ready, continue with the project-specific setup steps in this repository (dependencies, configuration, and running the app).
